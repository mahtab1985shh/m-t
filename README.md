# Founder Roadmap — Mahtab & Tom

یک فضای کاری فارسی برای تبدیل نقشه راه بنیان‌گذاران به مسیر اجرایی قابل پیگیری. این محصول بر پایه سند `Founder Roadmap Executive Version 1.0` ساخته شده است.

## امکانات

- ۱۳ فاز از هم‌راستایی بنیان‌گذاران تا رشد بلندمدت
- توضیح معنی و هدف هر فاز
- الزامات، اقدامات، مسیر مطالعاتی، خروجی‌ها و معیارهای موفقیت
- ثبت پیشرفت هر مورد در دیتابیس D1
- دفتر خاطرات و تجربه‌ها با امکان بارگذاری عکس در R2
- داشبورد واکنش‌گرا با زبان بصری الهام‌گرفته از monday.com

## اجرا

```bash
pnpm install
pnpm dev
```

## ساخت و آزمون

```bash
pnpm build
pnpm test
```

دیتابیس از طریق migrationهای پوشه `drizzle` ساخته می‌شود. تنظیمات میزبانی در `.openai/hosting.json` قرار دارد.
